Overview of the course

Programming an IoT requires a deep skill set that includes modifying physical processors, as well as programming wireless devices. Whether you’re interested in home automation, or microprocessor programing using products like Arduino or Raspberry Pi, E-learn has a course for you.

What You'll learn

  • Learn how to Independently Design, Code and Build IOT products.
  • Build 3 IOT products in the course
  • Learn how to connect to cloud IOT Platforms, Persist Data, Program Triggers and more
  • Learn to work with Micro controllers (Arduino Uno, Nano, NodeMCU), Sensors , Relays, Displays, Keypads, work with main (220/110) and moreHow not to manupulate data

Course Content

  • Basics Of IOT
  • IOT Used In Smart Watches
  • Characteristics and Levels of IOT devices - I am interesting!
  • IOT/communication Protocols - Hola!
  • Learn how to connect to cloud IOT Platforms, Persist Data and more
  • Learn to work with Micro controllers (Arduino Uno, Nano, NodeMCU), Sensors , Relays, Displays, Keypads, work with main and more
